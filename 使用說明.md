## 🚀 如何使用

1. **執行程式**
   - 直接雙擊 `ESEManager.exe`
   - 不需要安裝 Python 或任何依賴

2. **首次使用**
   - 程式會自動檢查資料庫更新
   - 可以在介面中搜尋和下載 TJA 譜面

3. **設定**
   - 下載目錄：可在介面中設定
   - 語言：支援繁體中文、英文、日文
   - 並行下載數：1-5 個同時下載

## 📋 功能特色

- ✨ 現代化 Liquid Glass 風格介面
- 🔍 支援日文歌名搜尋
- 📁 自動保持目錄結構
- ⚡ 支援並行下載
- 🌐 多語言介面（中文、英文、日文）
- 🔄 自動檢查資料庫更新
- 💾 自動儲存設定

## 📌 注意事項

1. **資料庫檔案**
   - 請勿刪除 `ese_songs.db` 和 `ese_local.db`
   - 這些檔案包含所有歌曲資訊

2. **網路連線**
   - 下載譜面需要網路連線
   - 程式會自動連接到太鼓譜面伺服器

3. **防毒軟體**
   - 部分防毒軟體可能會誤報，請加入白名單

## 🔧 進階使用

如需修改程式或重新打包：
1. 前往 `archive/source_code/` 找到原始碼
2. 修改後使用 `archive/build_files/` 中的 spec 檔重新打包
3. 指令：`python -m PyInstaller ese_gui_modern.spec --clean`
